Pull request Pasos

Pasos para hacer un pull request

  1. Entrar a tu cuenta de GitHub.
  2. Buscar el repositorio al que vas a contribuir (https://www.github.com/beztao01/niagara) o en la búsqueda dentro de GitHub tipeas: beztao01/beztao01/niagara.
  3. Le das click al botón fork ( Video 1:15 ).
  4. Nombras a tu repositorio, recomendable es que sea el mismo que el del repositorio original.
  5. Le das click al botón Create fork.
  6. Se crea en tu perfil una copia del repositorio.
  7. Buscas el icono de la pluma, entras al index.html desde gitHub.dev.
  8. Le das permisos a la extensión repositorio de Git Hub.
  9. Seleccionas tu cuenta.
  10. Agregas la línea que sea de tu interés.
  11. Le das click al icono de control de código fuente. ( Video 2:43 )
  12. Tipeas un mensaje acorde al cambio que hiciste.
  13. Click en el botón confirmar y guardar cambios.
  14. Le das click al botón pull request.
  15. Click al botón New pull request.
  16. Click al botón Create Pull request.
  17. En add a description, agregas un mensaje explicando que cambio con el que contribuyes.
  18. Click en el botón Create pull request.
  19. Checas que no tengas conflictos con el base branch.
  20. Termina tu contribución, solo esperas que Sandoval te apruebe el Pull request y que se cierre tu pull request.
